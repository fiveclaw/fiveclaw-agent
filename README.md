## What is this?

`fiveclaw-agent` is a local [MCP](https://modelcontextprotocol.io) server that runs on your machine alongside your FiveM server. It gives your AI client (Claude Code, Cursor, Windsurf, etc.) direct access to your server's files, logs, MySQL database, SSH, and txAdmin вҖ” and connects it to the FiveClaw cloud platform for FiveM-specific AI intelligence.

**Install once. Works everywhere you code.**

---

## Quick Install

```bash
pip install fiveclaw-agent
```

You'll need a [FiveClaw account](https://fiveclaw.xyz) and an API key from your [dashboard](https://fiveclaw.xyz/dashboard/keys).

## Local Tools

These run entirely on your machine.

| Tool | What it does |
|------|-------------|
| рҹ“Ӯ `repomap_generate` | Build a map of all resources in your server |
| рҹ”Қ `tool_search` | Search Lua/JS files across your resources |
| вң… `tool_syntax_check` | Check Lua syntax (embedded LuaJIT вҖ” no external binary needed) |
| рҹ“Ӣ `read_latest_logs` | Tail FXServer and resource logs |
| рҹ—ғ `tool_mysql_query` | Run queries against your FiveM MySQL database |
| рҹ–Ҙ `tool_server_control` | Start, stop, restart the FXServer via txAdmin or custom panel |
| рҹ”Ң `tool_resource_control` | Start/stop/restart individual resources |
| рҹ“Ў `tool_server_console` | Send console commands |
| рҹ”‘ `tool_ssh_run/ls/read/write` | Full SSH access to your remote server |
| рҹҡҖ `deploy_resource` | Deploy a resource directly to production |
| рҹ§  `context_remember` | Store persistent notes across AI sessions |
| в„№пёҸ `tool_platform_info` | Show configured OS, paths, and enabled services |
